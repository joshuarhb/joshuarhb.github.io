# Joshua Hamilton-Brown - Portfolio Website

A lightweight static portfolio site hosted on GitHub Pages.

## Live Site

https://joshuarhb.github.io

## Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages hosting

## Local Development

No build tool is required.

1. Clone the repository.
2. Open `index.html` directly in your browser, or serve locally with a static server.

Example:

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080.

## CV Download

The resume section includes a direct PDF download link:

`assets/files/joshua-hamilton-brown-cv.pdf`

Replace that file with your latest CV to keep the download current.

## Deployment

Use GitHub Pages with deploy-from-branch on `main` and root (`/`).
